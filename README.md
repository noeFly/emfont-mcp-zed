# emfont MCP Zed

Module Context Protocal Server for emfont.

## Install

1. Open [Zed](https://zed.dev).
2. Press `Ctrl+Shift+X` to open "Extension" window.
3. Search and install "emfont MCP".
4. That's it! Have fun!

## Development

1. Clone this project.
    ```sh
    $ git clone https://github.com/noeFly/emfont-mcp-zed.git
    ```

2. Install Rust via rustup

    https://rust-lang.org/tools/install/

3. Happy Coding!
